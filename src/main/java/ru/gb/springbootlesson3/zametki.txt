GET  /note/{id}   - получить заметку по id
GET  /note/user/{id}  - получить все заметки данного автора

PUT  /note/{id}   - изменить заметку с данным id       
POST /note        - добавить заметку
DELETE /note/{id} - удалить заметку с данным id

GET  /user/{id}   - получить автора по id
PUT  /user/{id}   - изменить автора с данным id       
POST /user        - добавить автора
DELETE /user/{id} - удалить автора с данным id

GET  /note?tag=...   - получить заметки c тегом (представляю тег как поле типа String: пример "работа","дом","сделать","важно"
